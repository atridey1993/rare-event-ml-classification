# Rare-Event ML Classification in Noisy, Imbalanced Data

## Overview
This project builds machine learning models for detecting rare signals in noisy and highly imbalanced datasets. It demonstrates practical skills in supervised learning, feature engineering, model comparison, and robust evaluation.

## Methods
- Data preprocessing
- Feature engineering
- Baseline classifier
- XGBoost
- Artificial Neural Network
- Evaluation with ROC-AUC, precision, recall, F1-score

## Skills Demonstrated
- Python
- Machine learning
- Imbalanced classification
- Feature engineering
- Model evaluation
- Data pipelines
